# Pingu

A simple discord bot useful for farming pings.

## Setup

- Simple create a new file in your root directory called `config.json` then fill out the information below.

```json
{
	"token": "",
	"role": "",
	"channels": [],
	"interval": 1000,
	"ratelimit": 10000
}
```

Then just run `npm install` and `npm start`.
